========================================================
SECTION 1 — AUTHENTICATION SYSTEM (PRODUCTION-READY)
========================================================

Implement a complete auth system with:

1. Email + Password Login
   - bcrypt password hashing (min 12 rounds)
   - JWT-based session (access + refresh tokens)
   - Secure HTTP-only cookies
   - Rate limiting on login attempts
   - Account lockout after 5 failed attempts

2. Forgot Password Flow
   - POST /auth/forgot-password
   - Generate secure reset token (crypto random, expires in 1 hour)
   - Store hashed token in DB
   - Send reset link (log to console if no email service)
   - POST /auth/reset-password validates token and updates password

3. Forgot Username (email lookup)
   - POST /auth/forgot-username
   - Returns masked email confirmation
   - Sends username via email/log

4. Google OAuth (no API key mode fallback)
   - If GOOGLE_CLIENT_ID present → use OAuth
   - If not → simulate SSO login with verified email domain
   - Always log auth source: "google" | "password"

5. Session Security
   - Rotate refresh tokens
   - Invalidate sessions on password change
   - Track login IP + device fingerprint
